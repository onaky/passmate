# PRD: PassMate MVP

## 개요
범용 자격증 학습 보조 PWA. 사진 촬영 → AI 분석 → 게임형 암기 퀴즈 3단계 루프.

## 사용자 스토리

### US-01: 온보딩
> 나는 수험생으로서, 내가 준비 중인 자격증을 선택하여 맞춤형 AI 해설을 받고 싶다.
- AC: 자격증 목록에서 선택 가능
- AC: 선택 후 로컬에 저장되어 다음 방문 시 유지

### US-02: AI 스캔
> 나는 수험생으로서, 교재 사진을 찍으면 AI가 즉시 해설해 주기를 바란다.
- AC: 카메라 촬영 또는 갤러리에서 이미지 선택 가능
- AC: 30초 이내 AI 분석 완료
- AC: Summary / Mnemonic / Keywords / RelatedConcepts 포맷으로 결과 출력

### US-03: 카드 저장
> 나는 수험생으로서, AI 분석 결과를 내 암기장에 저장하고 싶다.
- AC: 저장 버튼으로 IndexedDB에 영속화
- AC: 오프라인에서도 카드 조회 가능

### US-04: 퀴즈 진행
> 나는 수험생으로서, 저장한 카드를 게임처럼 퀴즈로 풀고 싶다.
- AC: 망각 곡선 알고리즘으로 복습 순서 결정
- AC: 정답 시 콤보 카운터 증가 및 파티클 애니메이션
- AC: 오답 시 즉시 재학습 카드 저장
- AC: 경험치 / 레벨업 시스템

### US-05: 대시보드
> 나는 수험생으로서, 내 학습 현황과 암기장을 한눈에 보고 싶다.
- AC: 전체 카드 수, 마스터 카드 수, 학습 시간 표시
- AC: 자격증별 필터링

## 비기능 요구사항
- 모바일 375px 기준 완벽 작동
- PWA: 홈 화면 추가, 오프라인 캐싱
- Lighthouse PWA 점수 90+
